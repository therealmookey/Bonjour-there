🏈 About This Scorecard

What is this?

This is a football-style scorecard for World of Warcraft guild members. It shows character stats in a visual, easy-to-read format.

Perfect for tracking guild progression and friendly competition!

How it works

The app uses the Blizzard API to fetch your guild roster and character data. It displays:

  • 🏆 Item Level - Best gear score
  • 📈 Achievement Points - Total achievements
  • 👥 Character Class & Race - Visual indicators
  • ⭐ Rankings - 🥇🥈🥉 for top 3

Setup Instructions

  1. Get API credentials from developers.battle.net
  2. Add your Client ID & Secret to config.js
  3. Update config.js with your guild info
  4. Push to GitHub and enable Pages

Tech Stack

  • 🌐 HTML + CSS - Sports-themed styling
  • ⚡ JavaScript - API calls and interactivity
  • 🎮 Blizzard API - Guild and character data
  • 🐙 GitHub Pages - Free hosting
